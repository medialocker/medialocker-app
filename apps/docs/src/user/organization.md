# Tags, Categories & Search

MediaLocker provides rich metadata features to organize your media beyond folders.

## Tags

Tags are free-form labels you can attach to any media file. Use them to mark:

- Content types: `raw`, `rough-cut`, `final`, `b-roll`
- Projects: `client-a`, `project-x`, `social-media`
- Status: `review`, `approved`, `archive`

### Adding Tags

1. Open a media item in the detail view.
2. Scroll to the Tags section.
3. Type a tag name and press Enter.
4. Existing tags autocomplete as you type.

### Filtering by Tags

Use the tag filter in the Media Library or the Search page to find files with specific tags.

## Categories

Categories are predefined, hierarchical groupings. Unlike free-form tags, categories have a fixed vocabulary.

Default categories:
- `Rough Cut`
- `Final`
- `Draft`
- `Raw`
- `Archive`

Custom categories can be configured in your organization settings.

### Category vs Tag

| Feature | Tags | Categories |
|---|---|---|
| Creation | Anyone can add | Admin-configured |
| Format | Free text | Enum values |
| Use case | Flexible labeling | Structured workflows |

## Search

The **Search** page provides full-text and faceted search:

1. Enter a search query to match filenames, tags, and metadata.
2. Use the **Faceted Filters** sidebar to narrow results by:
   - File type (image, video, audio, document)
   - Tags
   - Categories
3. Results update as you type and filter.

### Search Syntax

- `tag:rough-cut` — Filter to files with a specific tag
- `type:video` — Filter by MIME type group
- `bucket:my-bucket` — Search within a specific bucket
