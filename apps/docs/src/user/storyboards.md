# Storyboards

Storyboards are ordered sequences of media clips for timeline-based organization. Think of them as playlists or shot lists for visual content.

## What Are Storyboards?

A storyboard contains:
- **Clips** — Media files placed in a specific order
- **Timeline** — The ordered sequence of clips
- **Metadata** — Name, description, and organization info

Use cases:
- **Video editing**: Organize clips in the order they'll appear in a timeline
- **Photo shoots**: Sequence selects from a session
- **Presentations**: Order slides and media for a deck
- **Storytelling**: Arrange visual assets to tell a narrative

## Creating a Storyboard

1. Navigate to **Storyboards** in the sidebar.
2. Click **Create Storyboard**.
3. Enter a name and optional description.
4. Click **Create**.

## Managing a Storyboard

### Adding Clips

1. Open the storyboard detail view.
2. Click **Add Clip**.
3. Search and select media files.
4. Clips are added to the end of the timeline.

### Reordering Clips

1. In the storyboard detail view, drag and drop clips to reorder.
2. Or use the up/down arrow buttons on each clip.

### Removing Clips

1. Click the remove (X) button on any clip.
2. The clip is removed from the timeline but the media file is not deleted.

### Clip Properties

Each clip in a storyboard retains:
- Reference to the original media file
- Order position in the timeline
- Duration metadata (for video/audio)

## Deleting a Storyboard

Deleting a storyboard removes the timeline and all clip associations but does NOT delete the underlying media files.
