# Media Library

The Media Library is your central hub for browsing, viewing, and managing all media files.

## Browsing Media

Navigate to **Media Library** in the sidebar to see a grid view of your files. Each card shows:

- Thumbnail preview (when available)
- Filename
- File size
- MIME type

## Viewing Media

Click any media item to open the detail view. The detail view includes:

### Media Viewer
- **Images**: Full-resolution preview with zoom
- **Video**: Embedded video player with controls
- **Audio**: Audio player with waveform
- **Documents**: Text preview (PDF via embedded viewer)

### Metadata Panel
- Filename, size, MIME type
- Dimensions (images/video)
- Duration (video/audio)
- Upload date

### Actions
- **Download** — Download the original file
- **Delete** — Permanently remove the file
- **Add to Set** — Add this file to an existing or new set
- **Add to Storyboard** — Add this clip to a storyboard timeline

## Filtering

Use the filter bar at the top of the media library:

- **Search** — Full-text search by filename
- **Type filter** — Filter by file type (Images, Videos, Audio, Documents)
- **Tag filter** — Filter by assigned tags
- **Date range** — Filter by upload date
- **Faceted sidebar** — Available in the Search page for advanced filtering

## Infinite Scroll

The media library loads more items as you scroll. Results are paginated server-side for performance.
